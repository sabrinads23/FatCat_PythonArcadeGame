# Fat Cat | Python Arcade Game

Jogo acadêmico desenvolvido em Python com foco em lógica de programação, estruturação de código e desenvolvimento de aplicações interativas.

O jogador controla um gato que deve coletar comidas enquanto evita objetos ruins, utilizando sistema de pontuação, controle de vidas e temporizador.

## 💻 Tecnologias Utilizadas

* Python
* Pygame

## 🎮 Funcionalidades

* Sistema de pontuação
* Controle de vidas
* Temporizador
* Interação dinâmica com objetos
* Movimentação do personagem

## 📚 Objetivo do Projeto

Este projeto foi desenvolvido com o objetivo de praticar:

* Lógica de programação
* Organização e estruturação de código
* Manipulação de eventos
* Desenvolvimento de jogos 2D
* Interação entre elementos na tela

## 🚀 Aprendizados

Durante o desenvolvimento do projeto foram aplicados conceitos fundamentais de programação, resolução de problemas e criação de aplicações interativas utilizando Python.
