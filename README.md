# Fat Cat | Python Arcade Game
Projeto acadêmico desenvolvido em Python com foco em lógica de programação e estruturação de código. O jogo consiste em controlar um gato que deve coletar comidas enquanto evita objetos ruins, utilizando sistema de pontuação, controle de vidas e temporizador. O projeto foi desenvolvido para praticar conceitos de programação, organização de código e interação com elementos na tela.
